= ${artifactId}-ui

Dieses Projekt wurde mit create-react-app erzeugt.
* Anleitung: https://github.com/facebookincubator/create-react-app/blob/master/packages/react-scripts/template/README.md

== How to: Build
* Node.js muss installiert sein.
* .npmrc-Datei muss ins Benutzerverzeichnis kopiert werden

[source,bash]
npm run build

== How to: Start

=== Frontend im Entwicklungsmodus
Änderungen an .js Dateien werden sofort beim speichern sichtbar.
Benötigt ein lokal installiertes nodejs https://nodejs.org/

im Ordner src/${artifactId}-ui folgenden Befehl
[source,bash]
npm start

der Befehl öffnet automatisch einen Browser mit der Anwendung.
